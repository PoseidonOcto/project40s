How to build the project:

Pre-requisites: you must have Node and npm installed.

cd [directory you wish to build in]
git clone https://github.com/PoseidonOcto/Project40S.git
cd Project40S
npm install
npm run build

Now, load the extension in google chrome as shown here:
https://betterprogramming.pub/creating-chrome-extensions-with-typescript-914873467b65#:~:text=5.%20Load%20the%20extension
